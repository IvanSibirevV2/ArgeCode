Пространство имен System.VConsole
Папка VirtualConsole содержит последнюю версия надстройки консоли System.VConsole.Config.p_VirtualConsole
Глубокое копирование для VirtualConsole не предусматривается
Разнае экземпляры одной консоли предпологается использовать в тестовых функциях, интерфейсах, и модульных тестах.

При создании удаленных консолей экземпляры будут отличаться шаблонными лямбдами.
